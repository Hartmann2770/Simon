# Virtual Memory Palaces: Immersion Aids Recall

## Citation

Krokos, Eric, Catherine Plaisant, and Amitabh Varshney. "Virtual Memory Palaces: Immersion Aids Recall." *Virtual Reality* 23 (2019): 1–15. https://doi.org/10.1007/s10055-018-0346-3

[Read PDF in Zotero](https://www.zotero.org/esaitch/items/E4P2XFQV/attachment/U23TWGFR/reader)

## Key Findings

1. Participants using an immersive VR head-mounted display (HMD) to navigate a virtual memory palace recalled significantly more items (8.8% improvement) than those using a traditional desktop display.
2. The improvement was most pronounced for participants with low prior experience of memory palace techniques — novices benefit more from immersion than experienced mnemonists.
3. The vestibular and proprioceptive cues available in HMD-based VR (head-tracking, body orientation) appear to contribute meaningfully to spatial encoding of memory.
4. The study is among the first to operationalise the classical memory palace (method of loci) technique in a controlled VR environment and quantify the immersion benefit.
5. Desktop VR users also outperformed a non-VR 2D display baseline, suggesting a continuum of spatial presence effects on recall.

## Methodology

Controlled experiment (n = 40 participants). Three conditions: HMD VR, desktop VR (same 3D environment, no head-tracking), and a 2D image slideshow. Participants memorised a sequence of items placed in a virtual environment. Recall tested immediately after encoding. Spatial presence measured via questionnaire.

## Relevance

This paper directly bridges two domains central to this wiki: the classical [memory palace / method of loci](../concepts/spaced-repetition.md) mnemonic tradition and modern [virtual reality in education](../concepts/virtual-reality-in-education.md). The finding that immersion enhances recall via spatial cues supports [spatial presence](../concepts/spatial-presence.md) as a learning mechanism. Also connects to [retrieval practice](../concepts/retrieval-practice.md) literature, since the memory palace is itself a structured retrieval strategy.
